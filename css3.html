To perform calcuation with all the binaries we use ALU (Arithmetic and logical unit)

This is the first ALU built by INTEL
The output of the ALU needs to be storage in memory and its called RAM(Random access memory)
We have a CPU(Central processing unit) which connects to the RAM to get the data stored in memory and gets instructions from the instructions register and passed it to the ALU to perfom operations and as per the instructions recieved , stored it in another register. It follows the fetch/decode/execute phase . The speed with which it processes each cycle is handeled by the program counter.
